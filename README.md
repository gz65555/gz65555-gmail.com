# sao-sample

> my stylish SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao etu my-project
```

### From git

```bash
sao husong/sao-etu my-project
```
